# Contributing to Listify Backend

Thanks for your interest in contributing! Please follow these guidelines to keep the project consistent and easy to review.

## Branches
- Create a branch named `feature/<short-description>`, `fix/<short-description>`, `docs/<short-description>`, or `chore/<short-description>`.

## Pull Requests
- Open a pull request with a clear description of the change and why it's needed.
- Link to relevant issues and include screenshots or sample requests when applicable.

## Tests & Quality
- Add tests for new functionality and fix existing tests if necessary.
- Run `rubocop` to address style issues before opening the PR.
- The repository includes security scans (Brakeman) and test runs in CI — ensure changes pass those checks.

## Code Review
- Be responsive to review comments and make small, focused commits where possible.

## Thank you!
We appreciate your contributions — they help us maintain a robust, secure API for everyone.
